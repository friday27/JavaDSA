# Bit Manipulation

### Table (10-based to 2-based)
0   -> 0000,    5   -> 0101,    10  -> 1010,    15  -> 1111
1   -> 0001,    6   -> 0110,    11  -> 1011,    16  ->10000
2   -> 0010,    7   -> 0111,    12  -> 1100
3   -> 0011,    8   -> 1000,    13  -> 1101
4   -> 0100,    9   -> 1001,    14  -> 1110

### Basics
* ` ^ ` means XOR (same -> 0, different -> 1)
* ` ~ ` is NOT (0 -> 1, 1 -> 0)
* ` >> ` means RIGHT SHIFT
