# Stack

A stack is a one-ended linear data structure which models a real world stack by having 2 primary operations, namely push and pop.

### Complexity Analysis

        +-----------------+
        | Push   | O(1)   |
        +-----------------+
        | Pop    | O(1)   |
        +-----------------+
        | Peek   | O(1)   |
        +-----------------+
        | Search | O(n)   | <- Still takes linear time
        +-----------------+
        | Size   | O(1)!! |
        +-----------------+
